Bar Indicator

A jQuery plugin to visualize numeric data

This is a demo page that shows how you can use barIndicator plugin.

In order to implement it in your site, just copy the barIndicator folder with all it's contents and make links to the bi-style.css and jquery-barIndicator.js files

Don't forget to include jQuery before the plugin.

All you need is a simple html element like a span or a div

HTML <head>
 <link href="../barIndicator/css/bi-style.css" rel="stylesheet" />
 <script src="../jQuery.js"></script>
 <script src="../barIndicator/jquery-barIndicator.js"></script>
</head>
<body>
 ...
 <span id="bar">25</span>
 ...
</body>

And just initialize it!

jQuery <script>
 $('#bar').barIndicator();
</script>

Examples

For more information about options, methods and events, read the README file.

Horizontal (Default)

25
jQuery <script>
 $('#bar').barIndicator();
</script>

Vertical style

25
jQuery <script>
 var opt = { style:'vertical' };  $('#bar').barIndicator(opt);
</script>

Bar height horBarHeight

25
jQuery <script>
 var opt = { horBarHeight:30 };  $('#bar').barIndicator(opt);
</script>

Bar color foreColor

25
jQuery <script>
 var opt = {
  horBarHeight:30,
  foreColor:'#e25a48'
 };
 $('#bar').barIndicator(opt);
</script>

Bar holder color backColor

25
jQuery <script>
 var opt = {
  horBarHeight:30,
  foreColor:'#e25a48',
  backColor:'rgb(126,196,107)'
 };
 $('#bar').barIndicator(opt);
</script>

Label position horLabelPos

25 25 25
jQuery <script>
 var opt = {
  horLabelPos:'topRight' //'left'/'right'
 };
 $('#bar').barIndicator(opt);
</script>

Title horTitle

25
jQuery <script>
 var opt = {
  horTitle:'This is a sample title'
 };
 $('#bar').barIndicator(opt);
</script>
